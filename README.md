Este projeto é uma página de contagem regressiva criada para anunciar uma live ou evento que acontecerá em uma data específica. A estrutura combina HTML, CSS e JavaScript para montar um visual chamativo e uma lógica funcional.

A página mostra quatro elementos principais: dias, horas, minutos e segundos. Esses valores são atualizados automaticamente a cada segundo, usando uma função que calcula o tempo restante até a data do evento. A lógica converte o total de segundos em unidades de tempo, formata os números com dois dígitos e insere cada valor no lugar correto da interface.

O design usa como fundo uma imagem animada (um GIF do Sonic), que ocupa toda a tela e dá um toque mais dinâmico. Para garantir boa leitura, a página tem uma camada semitransparente por cima do fundo. Os textos usam a fonte Montserrat e crescem proporcionalmente ao tamanho da tela, garantindo responsividade.

Além disso, a página inclui música que começa automaticamente apenas quando o usuário interage com a tela, já que os navegadores bloqueiam autoplay por padrão. O som é ativado na primeira ação de clique ou movimento do mouse, e depois disso o listener é removido para evitar repetições.

No fim, existe um botão “Cadastre-se”, estilizado em preto com borda cinza, servindo como chamada para ação — mesmo que ainda não esteja conectado a nenhuma funcionalidade.

No geral, o projeto reúne visual moderno, animação, som e uma lógica bem construída para exibir uma contagem regressiva funcional. É uma base sólida que pode ser expandida para landing pages, anúncios de lives ou eventos especiais.
